# 自动化输入管道（Automated Input Pipeline）

让信息自动流入 [[entities/Obsidian]] 笔记库的自动化系统，降低手动整理的心智负担。

## 组成

| 管道 | 工具 | 功能 |
|------|------|------|
| 网页剪藏 | Obsidian Web Clipper | 一键保存网页/推文/视频字幕为 Markdown |
| 电子书划线 | Apple Books Highlights 插件 | 自动导入电子书高亮内容 |
| 语音闪念 | iPhone Shortcuts + Action Button | 一键录音 → 转文字 → 保存 Daily Notes |
| 视频字幕 | Web Clipper 阅读模式 | 边看边读字幕，点击时间戳跳转，一键导出 |

## 核心原则

如果存东西这件事很费劲，就不会坚持下去。自动化管道的目标是让信息流动**轻松到不需要手动整理**。
