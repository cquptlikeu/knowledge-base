# Go（Golang）

Google 开发的静态编译型编程语言，在本架构中作为后端唯一开发语言（版本 1.25）。

## 核心特征

- **高并发低开销**：goroutine 原生并发模型，单进程可调度数万并发连接，内存占用远低于 Java/Spring
- **静态编译单二进制**：所有依赖编译进单个可执行文件，无需运行时环境，容器镜像体积小
- **跨平台原生支持**：工具链原生支持交叉编译，可直接生成 ARM/MIPS 二进制
- **标准库完备**：内置 HTTP 服务器、加密算法、系统调用封装，第三方依赖少，供应链风险低

## 主要应用场景

- **NAS/边缘设备后端**：资源受限环境下的高性能服务
- **系统级编程**：需要与 Linux 内核特性（ACL、文件系统、网络协议栈）交互的场景
- **跨平台部署**：开发机直接生成目标架构二进制

## 与其他实体的关系

- **配合 [[entities/Gin]]**：Gin 是 Go 生态中性能最优的 HTTP 路由框架
- **配合 [[entities/SQLite]]**：使用 modernc.org/sqlite（纯 Go 实现），保持无 CGO 依赖
- **配合 [[entities/authd]]**：authd 全部用 Go 编写

## 在本架构中的选择理由

> Go 可将所有依赖编译进单个可执行文件，无需运行时环境，容器镜像体积小，冷启动快，部署无依赖地狱。
