# Llama

Meta（Facebook）开源的 LLM 系列模型。Llama 3（8B/70B/405B）是目前最流行的开源大模型之一，广泛用于本地化部署 Agent、代码审计、微调实验等场景。

## 核心特征

- **全开源**：权重开放，可本地部署，不依赖第三方 API
- **多规模**：从 8B（适合本地推理）到 405B（旗舰性能），按场景选择规模
- **生态活跃**：Ollama、vLLM、llama.cpp 等推理框架广泛支持
- **隐私友好**：本地部署不涉及数据外传，适合企业代码审计、敏感数据处理

## 主要应用场景

- **本地代码审计 Agent**：Llama 3 8B 部署在本地，扫描 Git 仓库代码的安全漏洞、性能问题
- **端侧推理**：量化后可在消费级硬件运行
- **微调实验**：开源特性使其成为模型微调的首选基座
- **隐私敏感场景**：医疗、金融、企业内网等不能使用云 API 的场景
